# Currículo online - Gabrielle Galante Pessoa Silva

Este pacote contém uma página de currículo profissional em HTML/CSS, pronta para publicar no GitHub Pages, Netlify, Vercel ou hospedagem comum.

## Arquivos

- `index.html`: página principal do currículo.
- `style.css`: estilos visuais, responsividade e impressão em PDF.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos `index.html` e `style.css` para a raiz do repositório.
3. Vá em **Settings > Pages**.
4. Em **Source**, selecione a branch principal e a pasta `/root`.
5. Salve. O GitHub vai gerar um link público em alguns minutos.

## Como gerar PDF

Abra o `index.html` no navegador e clique em **Salvar em PDF**.
Também é possível usar `Ctrl + P` e escolher a opção **Salvar como PDF**.
